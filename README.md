# PJ_Business-Monthly-Promotions
[2024.05 ~ 2024.07] 이커머스 매출 프로모션 전략 제안 프로젝트      
: 회사 매출 데이터 분석을 통한 월별 프로모션 상품 제안

## 프로젝트 배경
- 전사적 매출을 견인하는 핵심 카테고리·상품군과 월별 프로모션의 대표 키워드가 명확하지 않아 프로모션 효율화 필요
- Dacon 데이터를 활용한 "매출을 움직이는 실제 요인은 무엇인가?" 문제 재정의
- 시즌·제품 특성·분류 측면을 포함한 4개 가설 설정

## 핵심 내용
- 월별 상위 판매 제품 추출: 텍스트마이닝으로 주요 키워드·제품 특성 파악
- 월별 핵심 제품군의 판매 추이와 계절성 분석: 대·중·소 카테고리별 월평균 매출을 집계해 시기별 핵심 카테고리 정량 비교
- 브랜드 연관 키워드 분석: 연관량과 판매량·판매금액 간 상관관계 분석을 통해 핵심 의사결정 지표 도출
- 제품 특성 및 분류 파악: 월별 상위 판매 제품 텍스트마이닝/주기성 분석, 대·중·소분류별 판매량 비교 및 연관성 분석으로 핵심 제품군 식별
- 브랜드 연관 키워드 분석: 판매금액의 정규성 확인 후 대/중/소분류별 spearman, pearson로 상관계수, p-value 확인

## 주요 성과
- 특정 시기별 판매량 급증 제품 기반 **월별·시즌별 프로모션 후보군 도출**
- 카테고리·지표 파탕을 종합해 월별 주력 제품 제안 및 **사업 전략 요약 리포트 제작**
- IT 연합 동아리 **'TAVE' 13기 컨퍼런스 우수상** 수상

## 인사이트
- 데이터 너머의 상황과 맥락을 해석하는 과정에서 실제 비즈니스 환경에서 더 깊게 탐색하고, 지역 기반 사용자에게 의미 있는 의사결정을 돕고 싶은 욕구 형성

## 기술
* Pandas, R, SQL, Matplotlib, Seaborn, Konlpy
* Statsmodels (시계열 분석)

## 데이터 출처
* [Dacon] 온라인 채널 제품 판매량 예측 AI 온/오프라인 해커톤

## 파일 구조
- `3.1.R`: 제품 특성 측면(1) - 월별 주요 제품 특성 파악 (텍스트마이닝 & 주요 키워드 빈도 분석)
- `3.2.1.ipynb`: 제품 특성 측면(2) - 개별 제품에 따른 판매량 추이 파악
- `3.2.2.ipynb`: 제품 특성 측면(2) - 상위 5개 제품에 대한 주기성 분석
- `3.2.3.ipynb`: 제품 특성 측면(2) - 판매 추이를 갖는 제품 분석
- `3.3.ipynb`: 제품 분류 측면 - 제품 분류에 따른 제품 판매량 추이 파악 (대/중/소 분류별 판매량 추이 분석)
- `3.4.ipynb`: 브랜드 연관 키워드 언급량 측면 (판매량의 관계 파악, 1000이상 중가한 브랜드를 대상)
- `3.5.ipynb`: 판매량과의 상관관계 파악 측면 (정규성 확인 후 대/중/소분류별 spearman, pearson 상관계수, p-value 확인)
